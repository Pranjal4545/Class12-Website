1. Working with a Function

A function in C is a block of code that performs a specific task. It promotes code reusability and modular programming.

a. Define Function

A function is a group of statements that together perform a task. C has built-in library functions and allows user-defined functions.

b. Syntax of a Function

return_type function_name(parameter_list) {
    // body of the function
}

c. Types of Functions

  • Library functions: Predefined in C standard libraries (e.g., printf(), scanf()).
  • User-defined functions: Written by the programmer to perform specific tasks.

d. Components of a Function

  • Function Prototype: Declaration before main.
    Example: int add(int, int);
  • Function Call: Used to invoke the function.
    Example: sum = add(4, 5);
  • Function Definition: Actual body of function.
    int add(int a, int b) {
    return a + b;
}
  • Return Type: Specifies the data type of the value returned.

2. Categories of Functions (with Example)

i. Function with return type but no arguments

int getNumber() {
    int n;
    printf("Enter a number: ");
    scanf("%d", &n);
    return n;
}

int main() {
    int x = getNumber();
    printf("You entered: %d", x);
    return 0;
}

ii. Function with return type with arguments

int multiply(int a, int b) {
    return a * b;
}

int main() {
    int result = multiply(3, 4);
    printf("Result: %d", result);
    return 0;
}

iii. Function with no return type with no arguments

void greet() {
    printf("Hello, User!");
}

int main() {
    greet();
    return 0;
}

iv. Function with no return type with arguments

void display(int num) {
    printf("The number is: %d", num);
}

int main() {
    display(10);
    return 0;
}

3. Storage Classes in C

Storage class defines the scope, visibility, and lifetime of a variable.

  • auto – Default for local variables. 
    void show() {
    auto int x = 5;
    printf("%d", x);
}
  • extern – Used to declare global variables across files. 
    extern int x; // declared in another file
  • register – Requests storing variable in CPU register for speed. 
    void fastCount() {
    register int i;
    for (i = 0; i < 100; i++) {}
}
  • static – Retains value between function calls. 
    void counter() {
    static int count = 0;
    count++;
    printf("%d", count);
}

4. Recursive Function

A recursive function calls itself to solve a smaller problem.

Example: Factorial of a Number

int factorial(int n) {
    if (n == 0)
        return 1;
    else
        return n * factorial(n - 1);
}

int main() {
    int fact = factorial(5);
    printf("Factorial: %d", fact);
    return 0;
}

5. Passing Array to a Function

We can pass arrays to functions by specifying the array name.

Example: Print Elements of Array

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}

int main() {
    int nums[5] = {10, 20, 30, 40, 50};
    printArray(nums, 5);
    return 0;
}
