1. Working with a Function

A function in C is a block of code that performs a specific task. It promotes code reusability and modular programming.

a. Define Function

A function is a group of statements that together perform a task. C has built-in library functions and allows user-defined functions.

b. Syntax of a Function

return_type function_name(parameter_list) {
    // body of the function
}

c. Types of Functions

  • Library functions: Predefined in C standard libraries (e.g., printf(), scanf()).
  • User-defined functions: Written by the programmer to perform specific tasks.

d. Components of a Function

  • Function Prototype: Declaration before main.
    Example: int add(int, int);
  • Function Call: Used to invoke the function.
    Example: sum = add(4, 5);
  • Function Definition: Actual body of function.
    int add(int a, int b) {
    return a + b;
}
  • Return Type: Specifies the data type of the value returned.

2. Categories of Functions (with Example)

i. Function with return type but no arguments

int getNumber() {
    int n;
    printf("Enter a number: ");
    scanf("%d", &n);
    return n;
}

int main() {
    int x = getNumber();
    printf("You entered: %d", x);
    return 0;
}

ii. Function with return type with arguments

int multiply(int a, int b) {
    return a * b;
}

int main() {
    int result = multiply(3, 4);
    printf("Result: %d", result);
    return 0;
}

iii. Function with no return type with no arguments

void greet() {
    printf("Hello, User!");
}

int main() {
    greet();
    return 0;
}

iv. Function with no return type with arguments

void display(int num) {
    printf("The number is: %d", num);
}

int main() {
    display(10);
    return 0;
}

3. Storage Classes in C

Storage class defines the scope, visibility, and lifetime of a variable.

  • auto – Default for local variables. 
    void show() {
    auto int x = 5;
    printf("%d", x);
}
  • extern – Used to declare global variables across files. 
    extern int x; // declared in another file
  • register – Requests storing variable in CPU register for speed. 
    void fastCount() {
    register int i;
    for (i = 0; i < 100; i++) {}
}
  • static – Retains value between function calls. 
    void counter() {
    static int count = 0;
    count++;
    printf("%d", count);
}

4. Recursive Function

A recursive function calls itself to solve a smaller problem.

Example: Factorial of a Number

int factorial(int n) {
    if (n == 0)
        return 1;
    else
        return n * factorial(n - 1);
}

int main() {
    int fact = factorial(5);
    printf("Factorial: %d", fact);
    return 0;
}

5. Passing Array to a Function

We can pass arrays to functions by specifying the array name.

Example: Print Elements of Array

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}

int main() {
    int nums[5] = {10, 20, 30, 40, 50};
    printArray(nums, 5);
    return 0;
}

Functions

Introduction and Syntax

A function in C is a block of code that performs a specific task. Functions help in modular programming, reusability, and better debugging.

returnType functionName(parameters) {
   // body of the function
}

Purpose & Advantages

  • Code reusability
  • Simplifies debugging and testing
  • Improves program structure

Components of a Function

  • Function Prototype
  • Function Definition
  • Function Call
  • Return Statement

Types

Library Functions: Predefined functions like printf(), scanf().

User Defined Functions: Functions written by the programmer.

Passing Arguments

  • Call by Value
  • Call by Reference

Variable Scope

  • Local: Declared inside functions
  • Global: Declared outside functions

Storage Classes

  • Automatic
  • External
  • Static
  • Register

Function with Array Example

void display(int arr[], int n) {
   for(int i=0; i<n; i++)
      printf("%d ", arr[i]);
}

Recursive Function

int factorial(int n) {
   if(n==0) return 1;
   return n * factorial(n-1);
}

Advantages: Reduces code length, helps with problems like factorial, Fibonacci.

Disadvantages: Uses more memory (stack calls), may cause stack overflow.

Structure and Union

Structure

A structure in C groups variables of different data types under a single name.

struct Student {
   int id;
   char name[20];
   float marks;
};

Structure Size

The size depends on padding and alignment rules of the compiler.

Accessing Members

struct Student s1;
s1.id = 1;
strcpy(s1.name, "John");
printf("%d %s", s1.id, s1.name);

Nested Structure

struct Address {
   char city[20];
   int pin;
};
struct Student {
   char name[20];
   struct Address addr;
};

Array of Structures

struct Student students[10];

Passing Structure to Function

void printStudent(struct Student s) {
   printf("%s %d", s.name, s.id);
}

Union

A union allows storing different data types in the same memory location.

union Data {
   int i;
   float f;
   char str[20];
};

Comparison

Structure Union
Allocates memory for all members Shares memory for all members
All members can be used at once Only one member at a time

Pointers

Introduction

A pointer is a variable that stores the address of another variable.

int *ptr, a=10;
ptr = &a;

Pointer and Function

void update(int *n) {
   *n = *n + 10;
}

Call by Reference

Pointers are used to implement call by reference.

update(&a);

Pointer with Arrays

int arr[3] = {1,2,3};
int *p = arr;
printf("%d", *(p+1)); // 2

Advantages: Dynamic memory allocation, faster execution.

Disadvantages: Complex, risk of segmentation fault.

File Handling

Concept

Files are used to store data permanently. File handling in C provides mechanisms for sequential and random access.

File Handling Functions

  • fopen(), fclose() – open/close file
  • getc(), putc() – read/write character
  • fprintf(), fscanf() – formatted I/O
  • getw(), putw() – read/write integer
  • fgets(), fputs() – read/write string
  • fread(), fwrite() – block I/O
  • remove(), rename() – delete/rename file

Random Access Functions

  • fseek() – move file pointer
  • rewind() – reset pointer
  • ftell() – get current position

File Modes

  • r / r+ – read
  • w / w+ – write
  • a / a+ – append

Steps

  • Define File Pointer
  • Open File
  • Read/Write/Append
  • Close File

Typedef

The typedef keyword is used to create an alias for data types.

typedef unsigned int uint;
uint x = 5;
